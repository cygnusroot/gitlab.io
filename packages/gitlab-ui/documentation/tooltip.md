# Tooltip

<!-- STORY -->

## Usage
~~~js
<element
  v-gl-tooltip.${modifier}
  title="some tooltip text"
>
  ...
</element>
~~~

## Props

Tooltip uses the same props as [`<b-tooltip>`].

## Under the hood
Tooltip uses [`<b-tooltip>`] internally.

[`<b-tooltip>`]: https://bootstrap-vue.js.org/docs/directives/tooltip
